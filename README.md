# Jyotiksha Security Agency Pvt. Ltd.

A professional, responsive, and dynamic business website designed for **M/S Jyotiksha Security Agency Pvt. Ltd.** based in Bhadrak, Odisha. This website serves as the primary digital touchpoint for clients to explore security services, learn about the company, and easily get in touch for service inquiries.

## 🌟 Features

- **Modern Hero Section:** Engaging video background hero displaying the company's core values—Protection, Discipline, Trust.
- **Detailed Service Portfolios:** Individual, comprehensive pages for Security Guards, Gunman Services, Bouncer Services, and Body Guard Services.
- **Interactive & Responsive UI:** Clean navigation, smooth scrolling, and dynamic hover effects optimized for all devices (Mobile, Tablet, Desktop).
- **Floating Contact Button:** A sticky, hover-to-expand contact CTA globally available across the site for quick and easy inquiries.
- **Performance Optimized:** Static file architecture with clean, semantic HTML for blazing-fast load times.
- **SEO Ready:** Configured with meta tags and structured data (JSON-LD) for Local Business SEO in the Bhadrak/Odisha region.
- **Clean URLs:** Ready for instant deployment on Vercel with clean URL configurations included.

## 🛠️ Technologies Used

- **HTML5:** Semantic architecture and structure.
- **CSS3 (Vanilla):** Custom properties (CSS variables), Flexbox, CSS Grid, and responsive media queries.
- **JavaScript (ES6):** Interactive mobile hamburger menus and DOM manipulation.
- **GSAP (GreenSock):** ScrollTrigger utilized for smooth, modern reveal animations on scroll.

## 🚀 Deployment

This project is a static occupational website and is configured for seamless deployment to **Vercel**. 

**Steps to deploy via Vercel:**
1. Create an account or log in to [Vercel](https://vercel.com/dashboard).
2. Click **Add New** > **Project**.
3. Import this GitHub repository: `https://github.com/PranayaKD/Jyotiksha-Security-Agency-Pvt.-Ltd`
4. Leave the Framework Preset as **Other** (since it's a plain HTML/CSS/JS site).
5. Click **Deploy**.

*Note: The included `vercel.json` file automatically handles clean URLs, ensuring that users can visit routes like `/contact` instead of `/contact.html`.*
